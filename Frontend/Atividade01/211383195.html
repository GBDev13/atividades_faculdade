Lista de disciplinas por módulo

Módulo 51

Período: 19/10/2021 a 28/03/2022

  • GO - OPORTUNIDADES E RESULTADOS

    Na disciplina GO - Oportunidades e Resultados, foi comentado sobre construção de projeto de vida, identificação de oportunidades, preparação para oportunidades, conquista de oportunidades, gerenciamento de resultados, entre outros tópicos.

  • FUNDAMENTOS E ARQUITETURA DE COMPUTADORES

    Na disciplina de Fundamentos e Arquitetura de Computadores, foi comentado sobre a história da computação e suas gerações, códigos hexadecimais, conversão de binário para hexadecimal, entre outros tópicos.

  • GESTÃO DE PESSOAS E DESENVOLVIMENTO DE EQUIPES

    Na disciplina de Gestão de Pessoas e Desenvolvimento de Equipes, foi comentado sobre os conceitos de gestão de pessoas, desafios conteporâneos da gestão de pessoas, liderança e motivação, ambiente de trabalho e comunicação, estágios de grupos e desenvolvimento de equipes, entre outros tópicos.

Módulo 52

Período: 30/03/2021 a 06/06/2022

  • GO - PROJETO DE VIDA

    Na disciplina GO - Projeto de Vida, foi comentado sobre estado atual e estado desejado, equilibrio dos pilares da vida, espiritualidade e fé como valores pessoais, autorresponsabilidade, construção do projeto de vida, mapeamento do estado atual e desejado, estabelecimento do plano de ação, entre outros tópicos.

  • ALGORITMOS E LÓGICA DE PROGRAMAÇÃO I

    Na disciplina de Algoritmos e Lógica de Programação I, foi comentado sobre conceitos básicos da lógica da programação, estruturas condicionais, estruturas de repetição, estrutura de dados homogêneas e heterogêneas, sub-rotinas, entre outros tópicos.

  • ALGORITMOS E LÓGICA DE PROGRAMAÇÃO II

    Na disciplina de Algoritmos e Lógica de Programação II, foi comentado sobre os conceitos iniciais de programação e a estrutuda de um programa em C, estrutuda condicional, estruturas de repetição, vetores, strings, matrizes, structs, funções e arquivos, entre outros tópicos.

  • ESTRUTUDA DE DADOS I

    Na disciplina de Estrutuda de Dados I, foi comentado sobre ponteiros, pilhas, filas, listas dinâmicas, grafos, busca em grafos, entre outros tópicos.

  • FORMAÇÃO SOCIOCULTURAL E ÉTICA II

    Na disciplina de Formação Sociocultural e Ética II, foi comentado sobre ciência, tecnologia, meio ambiente, desenvolvimento sustentável, mapa mental, entre outros tópicos.

  • PROGRAMAÇÃO FRONT END

    Na disciplina de Front End I, foi comentado sobre os princípios básicos para desenvolvimento web, HTML 5, CSS 3, JavaScript e JQuery, entre outros tópicos.
